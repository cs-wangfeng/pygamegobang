# pygame_五子棋

#### 简介
该项目使用了python语言（编译环境为VS2019），并通过pygame库创造游戏的窗口、实现鼠标点击下棋、棋盘右侧实时显示下棋方、显示胜负信息、显示开局时间、播放背景音乐。
